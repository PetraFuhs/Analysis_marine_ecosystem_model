Organisation of this repository:

Section "Thesis": 
This section contains the thesis and its cover page as pdf files.

Section "Matlab":
		This section contains 5 MATLAB programs written & used 
		for section 5 in thesis. 
		All programs can be adjusted to parameter sets: parameters (including N_D) 
		and acceptable deviations to zero are found 
		at the top of the programs and can be varied.
		All programs produce a diary with notifications (e.g. if a root is negative 
		and hence a mathematical, but not an actual equilibrium). 
		All programs store results in tables. 
		These tables might be used for some extra figures. 
		All programs produce figures of interesting results 
		(at bottom of program code). 
		All programs are creted with Matlab Student Version V 7.10.0 (R2010a). 
		All programs are created by me (Petra Fuhs) to obtain information on the 
		equilibria of the NPZ model that are considered in section 5 of my thesis. 
		Remark: All programs free to use for non-commercial use without warranty.

		1.) Title: LLM1.m 
		Purpose: Testing stability of equilibrium LLM1, parameter set I, 
		with 3 values of N_D at 400 layer depths.

		2.) Title: LLM2.m
		Purpose: Calculating Equilibrium LLM2, parameter set I, 
		with 3 values of N_D at 400 layer depths, and testing stability. 
	
		3.) Title: LLM3.m
		Purpose: Calculating Equilibria LLM3 and LLM4, 
		parameter set I, with 3 values of N_D at 400 layer depths, 
		and testing stability. 
	
		4.) Title: QQM.m
		Purpose: Calculating Equilibria QQM2,QQM3 and QQM4, 
		parameter sets II and III, with 3 values of N_D at 400 layer depths, 
		and testing stability. 

		5.)	Title: QLM.m
		Purpose: Calculating Equilibria QLM2 and QLM3,
		parameter sets II and III, with 3 values of N_D at 400 layer depths, 
		and testing stability.
